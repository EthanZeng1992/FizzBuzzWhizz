## Require

  Ruby && Rspec

## Installation

```ruby
#Use Rvm to install Ruby :
#Install RVM :

$ curl -L get.rvm.io | bash -s stable
$ source ~/.bashrc
$ source ~/.bash_profile

#Install Ruby:

$ rvm install 2.1.1
 
# set ruby-2.1.1 as default
$ rvm use 2.1.1 --default
 

#Install Rspec :
$ gem install rspec
```
## Run
  $ ruby run.rb

## Test
  $ rspec spec/nums_spec.rb
